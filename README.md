# Car Price Prediction Using Machine Learning

## Objective
This project predicts car prices using machine learning techniques based on different car features from the dataset provided during the internship.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Features
- Data preprocessing
- Data visualization
- Model training
- Price prediction analysis

## Project Type
Machine Learning Internship Project

## Result
Built a machine learning model for predicting car prices using regression techniques.
